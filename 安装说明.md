# Chrome翻译插件安装说明

## 插件包信息
- **文件名**: chrome-translation-extension-fixed.zip
- **文件大小**: 约180KB
- **版本**: 1.0.1 (Service Worker修复版)
- **图标**: 已修复，包含所有必需尺寸
- **Service Worker**: 已修复兼容性问题

## Chrome扩展的两种格式

### 1. ZIP格式（当前提供）
- **用途**: 开发和测试阶段
- **安装方式**: 解压后通过"加载已解压的扩展程序"安装
- **优点**: 可以直接修改和调试
- **缺点**: 需要启用开发者模式

### 2. CRX格式（正式发布格式）
- **用途**: 正式发布和分发
- **安装方式**: 双击直接安装或通过Chrome Web Store
- **优点**: 用户友好，无需开发者模式
- **缺点**: 需要私钥签名，主要用于商店发布

## 安装步骤（ZIP格式）

### 1. 解压插件包
- 将 `chrome-translation-extension-fixed.zip` 解压到一个文件夹中
- 解压后会得到包含所有插件文件的文件夹

### 2. 在Chrome中加载插件

1. **打开Chrome浏览器**

2. **进入扩展程序管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者点击右上角三点菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展程序页面右上角，打开"开发者模式"开关

4. **加载解压的扩展程序**
   - 点击"加载已解压的扩展程序"按钮
   - 选择解压后的插件文件夹（包含manifest.json的文件夹）
   - 点击"选择文件夹"

5. **确认安装**
   - 插件会出现在扩展程序列表中
   - 确保插件状态为"已启用"

## 如何创建CRX格式（可选）

如果你需要CRX格式的插件包：

### 方法一：使用Chrome命令行
```bash
# 生成私钥（首次）
openssl genrsa -out key.pem 2048

# 打包为CRX
chrome --pack-extension=解压后的插件文件夹 --pack-extension-key=key.pem
```

### 方法二：使用在线工具
- 访问Chrome扩展打包工具网站
- 上传ZIP文件进行转换

### 方法三：通过Chrome开发者工具
- 在扩展程序页面点击"打包扩展程序"
- 选择插件文件夹和私钥文件

## 插件功能

### 主要特性
- **智能网页翻译**: 自动检测并翻译网页内容
- **选词翻译**: 选中文本即可快速翻译
- **学习模式**: 记录和管理学习过的词汇
- **离线功能**: 支持基本的离线翻译功能
- **多语言支持**: 支持多种语言互译

### 使用方法
1. **弹出窗口**: 点击浏览器工具栏中的插件图标
2. **选词翻译**: 在网页中选中文本，会自动显示翻译
3. **设置页面**: 右键点击插件图标 → 选项，进入设置页面
4. **词汇管理**: 在设置页面中管理学习过的词汇

## 注意事项

### 图标说明
- 当前使用蓝色方块作为临时图标
- 建议后续替换为专业设计的翻译主题图标

### 功能限制
- 部分高级功能可能需要网络连接
- 翻译质量依赖于所选择的翻译服务
- 开发版本可能存在一些类型错误（不影响基本功能）

### 故障排除
1. **插件无法加载**
   - 确保已启用开发者模式
   - 检查文件夹是否包含manifest.json文件
   - 确认图标文件存在且格式正确

2. **Service Worker问题**
   - 检查扩展程序页面中的"Service Worker"状态
   - 如果显示"无效"，请重新加载扩展
   - 查看控制台是否有"window is not defined"错误

3. **翻译功能异常**
   - 检查网络连接
   - 在插件设置中重新配置翻译服务

4. **界面显示问题**
   - 刷新页面重试
   - 重新启用插件
